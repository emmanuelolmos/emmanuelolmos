## Sobre mí
Me gusta construir interfaces limpias y APIs sólidas. Mantengo un estilo calmo-senior: decisiones simples, código legible, *todo en su lugar*.  
**Humor discreto:** prefiero que los bugs estén en los commits, no en producción 🫡.

- **Stack favorito:** Frontend con React + Tailwind, Backend con Node.js/Express (y Socket.IO si toca tiempo real).
- **Filosofía:** DX primero, pruebas manuales rápidas con Postman/Bruno/Hoppscotch y luego pulimos.
- **Herramientas diarias:** terminal, Git/GitHub, VS Code y Neovim (equilibrio zen).
